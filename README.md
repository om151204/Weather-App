# Weather App ☁️🌤️🌧️

A Java-based weather app that fetches real-time weather data using OpenWeather API.

## Features
- 🌡️ Temperature
- 🌬️ Wind Speed
- 🌅 Sunrise & Sunset Time
- 🌊 Height from Sea Level
- 💧 Humidity
- 🌦️ Weather Condition

## Technologies Used
- Java
- OpenWeather API
- JSON Parsing
- GUI (if applicable)
